## Setting up the dev environment

```bash
$ git clone https://github.com/stepney141/dagaz-new
$ cd dagaz-new
$ npm install
```

## Testing

This repository has unit tests based on [Jest](https://jestjs.io/) test suite. You can run them with Node.js `v14`.

```bash
$ npm test
```
